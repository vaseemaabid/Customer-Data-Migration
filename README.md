# Customer Data Migration and Validation System

## Project Overview
This project simulates customer data migration from a source system to a target system.

## Technologies Used
- Python
- SQL
- MySQL
- Excel
- Pandas

## Features
- Data Extraction
- Data Cleansing
- Data Validation
- Data Mapping
- Data Loading
- Reporting

## Workflow
1. Extract customer data from CSV.
2. Remove duplicate records.
3. Validate customer information.
4. Clean invalid records.
5. Load clean data into target system.
6. Generate reports.

## Skills Demonstrated
- SQL
- Data Migration
- Data Validation
- Data Cleansing
